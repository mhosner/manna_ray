Four Core Components

Command Router

Single entry point: /manna. Routes to skill execution, workflow management, context ops, or status queries.

Context Manager

Initializes project with scaffolded context files. Loads relevant context before each skill. Tracks staleness. Prompts for updates after skills produce new insights.

Pipeline Runner

Defines workflows as ordered skill sequences. Validates prerequisites before each step. Tracks progress. Handles the feedback loop (post-launch → discovery).

State Store

JSON file tracking active workflows, run history, output locations, and context checksums. Enables /manna status and dependency validation.
